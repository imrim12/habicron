Habits,
not cronjobs.

A tiny scheduler for randomized recurring tasks — one engine for Node, Vue, React, and your shell. Your jobs fire on a human rhythm — close to schedule, never robotically on the dot.

Node · Vue · React · CLI · tiny, zero‑dependency core
accurate by default, jittered on demand · no drift · SSR‑safe

habits.ts live 
import { useHabit } from 'habicron/vue'

// remind me to drink water every hour
useHabit(drinkWater, { every: '1h ± 8m' })

// check my cat every 2 hours
useHabit(checkCat, { every: '2h ± 20m' })

// a whole routine, with controls
const { nextRun, pause } = useHabit(routine, {
  controls: true,
  habits: [
    { every: '20s', jitter: ['3s', '5s'] },
    { times: 2, per: 'day', jitter: '2h' },
  ],
})

Examples

Four ways to use it

node 
import { createHabit } from 'habicron'

// poll a feed, roughly every 15 min
const job = createHabit(syncFeed, {
  every: '15m ± 2m',
})
react 
import { useHabit } from 'habicron/react'

useHabit(runAgent, {
  controls: true,
  habits: [
    { every: '2h ± 20m' },        // check the cat
    { times: 2, per: 'day', jitter: '90m' },
    { every: '3d', jitter: ['3h', '5h'] },
  ],
})
cli — run any command on a habit 
# stretch reminder, roughly every 10 minutes
$ habicron --every "10m ± 2m" -- notify-send "Stretch break"

# pull a repo a few times a day, nicely spread out
$ habicron --times 3 --per day --jitter 2h -- git pull

# scrape a page hourly, jittered so you don't look like a bot
$ habicron --every 1h --jitter 12m -- node scrape.js
vue — read it in your template 
<script setup lang="ts">
const { counter, nextRun, pause, resume } = useHabit(post, {
  controls: true,
  every: '20s ± 4s',
})
</script>

<template>
  <p>fired {{ counter }}× · next at {{ nextRun?.toLocaleTimeString() }}</p>
  <button @click="pause">Pause</button>
</template>

Web apps
